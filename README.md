# HR Attrition Analysis Dashboard

## Project Overview
An HR analytics tool designed to help organizations understand why employees leave. This dashboard tracks attrition across departments, age groups, and salary brackets.

## Key Features
* **Attrition Metrics:** Calculated Attrition Rate, Average Age, and Average Salary.
* **Demographic Analysis:** Breakdown of employee turnover by Gender and Education Field.
* **Job Satisfaction:** Correlation between job role, environment satisfaction, and retention.

## Technical Skills Used
* **DAX:** Developed measures for dynamic attrition percentages.
* **Data Modeling:** Created relationships between employee demographics and performance data.
* **Power BI:** Interactive filtering by Department (Sales, R&D, HR).

## Business Impact
* Identified that employees with lower "Work-Life Balance" scores have a 25% higher attrition rate.
* Targeted specific age groups (25-35) for retention programs based on high turnover data.
